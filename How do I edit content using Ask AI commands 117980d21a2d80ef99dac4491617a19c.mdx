# How do I edit content using Ask AI commands?

Paperguide offers a powerful feature that allows you to edit and generate content. AI-driven commands helps to enhance or modify it. With this feature, you can quickly improve your writing, adjust tone, simplify language, or even translate content. Here's how to use these AI commands for selected text:

[https://www.loom.com/embed/70f8ea2726e74a1b9d8b90ed68b827cd?sid=83d12108-136e-475b-b07b-16f19b9d72a2](https://www.loom.com/embed/70f8ea2726e74a1b9d8b90ed68b827cd?sid=83d12108-136e-475b-b07b-16f19b9d72a2)

Here's a step-by-step guide to using this feature:

At any time, you can **highlight** a section of **text** from Paperguide and edit it using Ask AI Commands. You can ask Paperguide to edit using a custom prompt or a complete set of preset functions:

- **Custom Prompt:** In the input field you can enter your own custom prompt to edit or modify your writing exactly how you need.
- **Improve Writing**: Refines your writing by enhancing readability, flow, and style.
- **Humanize (AI Bypass)**: Adjusts the text to make it sound more natural and less AI-generated.
- **Change Tone**: Adjusts the tone of the writing as you need.
- **Simplify**: Reduces complexity in the text, making it easier to read and understand.
- **Translate**: Allows you to translate the selected text into various languages for multilingual support.

1. Once the text is selected, a tool bar with few options will be displayed along the text selection.
2. Click on the **Ask AI** on tool bar. Then enter a custom prompt in the input field or select the **Edit** command you want from the dropdown.
3. Once Paperguide has generated your edited version, you will have the option to accept it to your paper try another suggestion, or delete it and try again with a different edit input.

**Example**

Choose AI commands based on the context of your writing. For example, use **Change Tone** when switching from academic writing to a more informal blog style.
