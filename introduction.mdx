## Core Features

You begin your research journey with Paperguide's intuitive interface. The platform supports uploading PDFs, connecting to databases like PubMed or Google Scholar, and importing from Zotero. AI analyzes content to extract key themes, relationships between studies, and potential gaps in literature.

## Use Cases

Researchers use Paperguide to accelerate literature reviews by summarizing hundreds of papers into actionable insights. Students benefit from guided writing prompts that align with citation styles like APA or MLA. Professionals in consulting apply it for competitive analysis, generating reports with embedded references.

<Tabs uid="fd51f934-c6c2-4c04-877a-428b32d51269">
  <Tab title="Academic Research" icon="book-open">
    Focus on thesis development with AI outlining tools.

    ```javascript
        // Example query for literature review
    const query = 'climate change impacts on agriculture';
    const results = await paperguide.search(query, { limit: 50, filters: { year: '>2020' } });
    console.log(results);
    ```
  </Tab>

  <Tab title="Professional Analysis" icon="bar-chart">
    Analyze market trends with integrated data visualization.

    ```python
    import paperguide as pg
    results = pg.search('AI in healthcare', filters={'domain': 'medicine'})
    print(results.summary())
    ```
  </Tab>
</Tabs>
