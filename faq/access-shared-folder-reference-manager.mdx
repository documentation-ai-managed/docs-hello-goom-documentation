# How can I access a folder shared with me in reference manager?

If you’ve been invited to a shared folder, you will receive an ✉️ **email invitation** from **Paperguide** to accept invite to **join shared folder**.

1. **Check Your Email**:

   - If you’ve been invited to a shared folder, you will receive an email invitation from Paperguide.

   - Even if you are not a Paperguide user and have been invited to shared folder, you can still access the folder by registering for an account.

   - To access the shared folder, you must log in or sign up using the **same email address** that **received the invitation**. This ensures you have the correct permissions to view or interact with the shared file.

2. **Accept/Reject Invitation:**



   - On clicking the **Accept/Open folder** button, you’ll be redirected to Paperguide to take action accordingly.

   - Once you accept the invitation, you can access the folder under **shared with me** section.

   - If you reject the invitation, you won’t be able to access it. And admin must re-invite you to join, if you want to join the folder.

3. **Access the Folder**:

   - Scroll down to “**Shared with me**” section under your folders and click on the folder that you were invited.

   - You will be able to access the folder based on the permissions assigned to you:

     - **Can View**: Allows you to **view** the folder, its papers, and any notes.

     - **Can Add Notes**: Provides viewing access and the ability to **add notes** to the papers.

     - **Can Edit**: Grants full access to **view, edit, add notes, and modify** the papers within the folder.
