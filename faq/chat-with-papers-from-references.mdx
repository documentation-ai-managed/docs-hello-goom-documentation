## Chat with Papers from References

This feature allows you to select the papers/folders from the reference manager and initiate a chat with the selected papers. Here, you can ask questions and chat after selecting papers.

<Video uid="1ad0b488-2891-43c9-b7e7-834d853d64ef" src="https://www.loom.com/embed/e9ffa1905668423293bf538057e73172?sid=393bfee4-9fcc-4193-9540-e30abe4a83bb" render-type="iframe" width="672" height="378" controls="true" allow-full-screen="true" style="width: 672px; height: auto; cursor: pointer;" />

1. Go to **Reference Manager** and select the papers that you’d like to chat by clicking on the check box before each paper’s title.

2. Then, click on "**Chat with Papers**" option displayed on the top header. This opens a chat interface for the selected papers.

3. You can now ask questions about the selected papers, and the AI will provide comprehensive answers from the content of all selected papers.
