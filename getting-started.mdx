## Account Setup

You start your Paperguide experience by creating a secure account. The platform requires basic information like email and password, ensuring quick onboarding. Once registered, you gain access to core tools for research management.

<Callout kind="tip">
  Use a strong password to protect your research data.
</Callout>

Follow these steps to get up and running:

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Navigate to the registration page. Enter your details and agree to the terms of service.
    
````javascript
    // Simulated signup
const userData = {
      email: 'researcher@example.com',
      password: 'securePass123'
    };
    // POST to /signup endpoint
    fetch('/api/signup', {
      method: 'POST',
      body: JSON.stringify(userData)
    });
````
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification email. Click the link to activate your account.
  </Step>
  <Step title="Initial Configuration" icon="settings">
    Set your preferences, such as default citation style and notification settings.
  </Step>
</Steps>

## Importing Your Library

You import existing references to build your library swiftly. Paperguide supports multiple formats for seamless migration from tools like EndNote or Mendeley.

<Tabs>
  <Tab title="From File" icon="upload">
    Drag and drop files into the import area. The system parses metadata and adds entries automatically.
    
````bash
curl -X POST /api/import \
  -H "Authorization: Bearer {token}" \
  -F "file=@library.ris"
````
  </Tab>
  <Tab title="From Database" icon="database">
    Connect to external sources like arXiv or JSTOR. Authorize access for direct pulls.
    
````python
from paperguide import Client
client = Client(api_key='{your_key}')
library = client.import_from('zotero', user_id='12345')
````
  </Tab>
</Tabs>

## First Search

Conduct your initial search to experience AI capabilities. Enter natural language queries, and Paperguide delivers curated results with summaries.

<Expandable title="Advanced Search Tips" default-open="false">
  Use operators like `AND` or `OR` for precise results. For example, `machine learning AND ethics` narrows focus.
</Expandable>

````javascript
// Example search implementation
const searchResults = await client.search({
  query: 'quantum computing applications',
  limit: 20,
  filters: { year: '>=2018', domain: 'physics' }
});
console.log(searchResults);
````

You receive results ranked by relevance, including abstracts and key excerpts. Save promising items to your library with one click.

If imports fail, check file encoding. For search issues, clear cache and retry. Contact support via the help center for personalized assistance.
